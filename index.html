mod-lang-js

A JavaScript™ Language Module for Vert.x

This is the Javascript language module used by mod-lang-dynjs, mod-lang-rhino, and soon mod-lang-nashorn. This module is not typically used directly, instead, mod-lang-rhino, mod-lang-dynjs, and mod-lang-nashorn all have a dependency on this module which provides the core Javascript API for Vert.x.
